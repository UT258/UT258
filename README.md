### 🏥 ClaimInsight360
> **Insurance Claims Analytics Platform**

A full-stack, distributed system built for high availability, security, and scalability.

**Architecture Highlights:**
- 🔧 **9 Microservices** behind a reactive API Gateway (Spring Cloud Eureka + WebFlux)
- 🛡️ **Stateless JWT Security** with 6 distinct business roles and 14 route-level authorization prefixes
- 🔄 **Cross-Service User Sync** via WebClient for consistent identity management
- 🖥️ **React 18 Frontend** — Redux Toolkit, polling-based notification UX, 12+ secure feature pages
- 📡 **Full Observability** via Zipkin distributed tracing, Spring Boot Actuator & Admin

**Stack:** `Java` `Spring Boot 3` `Spring Cloud` `WebFlux` `React 18` `Redux` `MySQL` `Docker` `JWT`

### 📓 Journal API
> **Secure Personal Journaling REST API**

A robust RESTful backend to securely manage, categorize, and analyze user journal entries.

**Feature Highlights:**
- 🔒 **Per-User Data Isolation** — strict ownership enforcement via Spring Security + JWT
- 🏷️ **Tag-Based Categorization** for organizing and filtering entries
- 🧠 **NLP Sentiment Analysis** — auto-classifies entries as positive / negative / neutral to track mood trends
- 📖 **Swagger/OpenAPI Docs** — fully documented and exploration-ready
- ✅ **JUnit Test Suites** for reliability and regression safety

**Stack:** `Java` `Spring Boot` `MongoDB Atlas` `Spring Security` `JWT` `JUnit` `Swagger`
