### 📌 Featured Engineering Work

<details open>
<summary><b>🏥 ClaimInsight360 – Insurance Claims Analytics Platform</b></summary>
<br>

> *A full-stack, distributed platform designed for high availability and secure data processing.*

- **Distributed Architecture:** Engineered **9 microservices** behind a reactive API Gateway (Spring Cloud Eureka, WebFlux) with role-based authorization across 14 route prefixes.
- **Identity & Security:** Designed stateless JWT-based identity management supporting **6 distinct business roles**, featuring cross-service user sync via WebClient.
- **Frontend & UX:** Developed a highly responsive React 18 frontend with Redux Toolkit, featuring a polling-based notification UX and 12+ secure feature pages.
- **Observability:** Implemented end-to-end tracing and real-time monitoring utilizing Zipkin, Spring Boot Actuator, and Spring Boot Admin.

🔗 **[View Repository](https://github.com/UT258)**
</details>

<details open>
<summary><b>📓 Journal API</b></summary>
<br>

> *A robust RESTful API built to securely manage, categorize, and analyze user entries.*

- **Core Stack:** Built with Spring Boot, MongoDB Atlas, Spring Security, and JWT.
- **Key Features:** Engineered strict per-user data isolation, tag-based categorization, and an integrated **NLP sentiment classification** system (positive/negative/neutral) to track mood trends over time.
- **Quality Assurance:** Fully documented via Swagger/OpenAPI and validated against rigorous JUnit test suites.

🔗 **[View Repository](https://github.com/UT258/journalApp)**
</details>
