## 📌 Featured Projects

### ClaimInsight360 – Insurance Claims Analytics Platform
- Architected **9 microservices** behind a reactive API Gateway using Spring Cloud Eureka and WebFlux with role-based authorization on 14 route prefixes.
- Designed JWT-based identity with **6 business roles** and cross-service user synchronization via WebClient.
- Built a scalable notification engine with scheduled fan-outs, per-user deduplication, and polling-based frontend UX.
- Developed a responsive React 18 + Redux Toolkit + Ant Design frontend with **12+ feature pages** and protected routes.
- Implemented end-to-end observability using Zipkin, Actuator, and Spring Boot Admin.
- 🔗 Repo: [UT258 on GitHub](https://github.com/UT258)

### Journal API
- Built a RESTful journaling API with Spring Boot + MongoDB Atlas supporting full CRUD with tag-based categorization.
- Secured endpoints with Spring Security + stateless JWT and strict per-user data isolation.
- Integrated NLP sentiment classification (positive/negative/neutral) for trend analysis.
- Documented API with Swagger and validated behavior with JUnit and Postman.
- 🔗 Repo: [journalApp](https://github.com/UT258/journalApp)
